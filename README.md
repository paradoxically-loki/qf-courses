# Quant Fin Courses at IITM
The repo is a collection of all QF courses that I did. The course are arranged semester-wise, and should contain notes/resources/exams/etc.

You may clone the repository by running
```bash
git clone https://github.com/paradoxically-loki/qf-courses.git
cd qf-courses
```

Raise issues, if any. Make PRs if you want to add something.
